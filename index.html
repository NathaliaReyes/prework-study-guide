HTML

  • The head element contains information about the webpage.
  • The head also contains information, or metadata, that helps the browser to render the page correctly.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • GITFLOW:
  • git add -A : stages all changes in your working directory for the next commit
  • git commit -m "message that we want to commit": for adding a commit message directly from the command line.
  • git pull origin main : it fetches and merges changes from the remote main branch into your local main branch.
  • git push origin branch-name : To push the changes from your local branch ('branch-name') to the remote repository ('origin').
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone [URL]: retrieve an entire repository from ahosted location via URL
  • git log: show all commits in the current branch's history

JAVASCRIPT

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
